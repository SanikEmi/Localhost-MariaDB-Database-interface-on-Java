# Localhost-MariaDB-Database-interface-on-Java
Instalar JDK 21 y añadir al PATH.
Aplicación en Java Netbeans para modificar bases de datos en SQL.
  Añadir archivo "mysql-connector-java-5.1.24-bin" al projecto como librería para solicitar la conexión a PHPMyAdmin.
  
  Modificar "nombreBD" para el nombre de la base de datos default para el argumento try-catch (en local IP).
